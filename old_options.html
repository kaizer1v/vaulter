Welcome to Vaulter

Vaulter is a safe way to store all your login details and can retrieve them without hassle, in your own format. Vaulter uses google sheet that you can configure to save your login details and retrieve it whenever you visit the website.

Getting started.

To get started with saving your login details, you can duplicate or copy this google sheet file and add your login details in columns specified. It has some predefined websites that you might have accounts in already with main columnsl ike the weblink of the login page of the website, along with the website name, username and password columns.

Note, the columns marked with (*) below are mandatory to have data for in order for Vaulter to run. The rest of the columns are for your reference and additional details. Here's a brief description of the columns in the sheet

  • - A reference name for you to maintain
  • - A weblink of the website for which you are storing the login details
  • - The username for the website
  • - Password for your website. Hint, you can use white text colour on the google sheet to make it invisible or hide this column
  • - Any notes or comments you want to save along with the login details
  • - Categorising websites for your reference

Vaulter uses this google sheet as your library of login details and retrieve as and when you request from the weblink you browse. See below configuration section to configure google sheets with vaulter

Google sheet configurations

In order for Vaulter to function, you'll need to provide the google sheet ID and a few column configurations incase you aren't using the default template structure provided above.

It is very important to note that whichever google sheet file you use, you'll need to set sharing permissions for the user you are logged into on Google chrome. You can share the file with restricted access.

Column Mapping

Provide the columns that map to your login details. Specify the index number of the column instead of the alphabet itself. For example, instead of `A` write `1` and for `B` write `2` and so on. Once you enter the required details, hit save.
