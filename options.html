Welcome to Vaulter

Vaulter is a safe way to store all your login details and can retrieve them without hassle, in your own format.

Readme first.

You can duplicate or copy this sample google sheet template and get started. It has some predefined websites that you might have accounts in already.

The sheet contains the following columns that are used to retrieve the details. Note, the columns with (*) are mandatory to have, without which the plugin won't function. The rest of the columns are for your reference.

  • - A reference name for you to maintain
  • - A weblink of the website for which you are storing the login details
  • - The username for the website
  • - Password for your website. Hint, you can use white text colour on the google sheet to make it invisible or hide this column
  • - Any notes or comments you want to save along with the login details
  • - Categorising websites for your reference
